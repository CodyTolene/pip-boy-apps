### Info

**Version:** 1.0.1

**Website(s):**

### Description

A classic 2048 game.

### Controls

- Use the LEFT WHEEL to shift the table Up / Down
- UPPER RIGHT WHEEL to shift Left / Right
- Push LEFT WHEEL Click to Reset
- Press TOP TORCH button to exit the game.
